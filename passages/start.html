如何为Ubuntu和其他Linux发行版添加开机音乐或管理开机启动项

在终端中输入以下指令：

gnome-session-properties

打开启动应用程序首选项的图形界面，点击添加，然后补充下述内容：

名称：Gnome Login Sound

命令：paplay /你所在音乐的路径

注释:（可不填）This is a programme to enable startup sound

添加后, 重启就行了。这个方法也可以用来管理其他启动项。

你也可以安装Tweak优化软件：

sudo apt-get install gnome-tweak-tool

在程序坞里找到工具-优化文件夹-优化（有些版本叫tweak）

左栏“开机启动程序“便可管理。
