Ubuntu Linux软件推荐（原生）

ubuntu的软件格式为 .deb，可使用应用商店下载的以倾斜标识。

1.文件解压缩

自带解压缩工具“归档管理器”。

2.视频播放器

VLC，从自带应用商店下载或从官方网站下载。也可打开终端，输入：
sudo apt-get install vlc

腾讯视频：官方网站

建议直接使用浏览器访问在线视频网站。

3.音乐播放器

VLC可以播放音频文件，自带的Rhythmbox Music Player也可以应付部分需求

LISTEN ONE（聚合平台）：官方网站

网易云Linux版本：官方网站

QQ音乐：官方网站

4.浏览器

见“关于我的博客”一栏

5.Office软件

WPS官方网站

永中Office（敬请期待）官网

Libre Office（部分发行版自带）官方网站

Open Office （部分发行版自带）官方网站

6.IDE集成开发环境

eclipse官方网站或者输入：sudo apt-get install eclipse

PycharmPycharm官网

7.文本编辑器

Microsoft Visual Studio Code官方网站

VIM 在终端中输入：sudo apt-get install vim

8.阅读软件

chm格式：Kchm Viewer，在终端中输入：
sudo apt-get install kchmviewer

如果中文乱码，请在『view』-『set encoding』更改编码类型

pdf格式：自带pdf查看器，或使用浏览器打开

pdf编辑：pdfeditor foxitsoftware

Calibre(全能阅读器），在终端输入：
sudo apt-get install calibre

9.视频会议软件

腾讯会议 官方网站

钉钉 官方网站

飞书 官方网站

10.多媒体编辑

GIMP（GNU图像处理程序）：
sudo apt-get install gimp

Shotcut(视频剪辑）： 官方网站

ffmpeg(开源音视频处理工具，只有命令行）：
sudo apt-get install gimp

QwinFF(上述工具的图形界面）：
sudo apt-get install qwinff

Da Vinci: 官方网站

11.虚拟机与模拟器：

vmware： 官方网站

麟卓 xdroid（安卓模拟器）: 官方网站

Virtual Box 官方网站

12.通讯软件

QQ for linux(不建议用）官方网站

微信（优麒麟版）官方网站

13.下载器

Free Download Manager 官方网站

Xtreme Download Manager 官方网站

Flareget Download Manager（15日试用期） 官方网站

以上需要配合浏览器插件使用，自行在浏览器扩展商店搜索

百度网盘 官方网站

14.游戏

Steam 官方网站

Watt Toolkit 加速器官方网站

MInecraft

15.其他工具

LAN drop（局域网传输文件）

新立得安装包管理： sudo apt-get install synaptic

Neofetch (在终端显示系统详情）： sudo apt-get install neofetch

Appimage Laucher(将.appimage程序安装到系统）：

sudo add-apt-repository ppa:appimagelauncher-team/stable

sudo apt-get update

sudo apt-get install appimagelauncher

(或者您也可以点击这个链接获取离线程序包（推荐）

Ventoy 启动盘制作工具（只需制作后把镜像丢里面）：Vnetoy官方网站
