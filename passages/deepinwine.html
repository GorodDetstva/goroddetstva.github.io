基于Deepin-Wine的Linux软件推荐

1.安装

首先，您必须在终端中输入如下指令，以添加DEEPIN-WINE的仓库。

wget -O- https://deepin-wine.i-m.dev/setup.sh | sh

自此以后，你可以像对待普通的软件包一样，使用apt-get系列命令进行各种应用安装、更新和卸载清理了。

2.常用的软件包名

以下列举了一些常用的软件包名。如要安装，您只需和其他软件一样，输入：

sudo apt-get install xxx

3.常见问题

1.图标问题

您需要重启电脑才能在程序坞（我也不知道那叫啥，反正在GNOME桌面就是那九个点）里看到。

2.微信启动不起来

其实我也没解决这个问题，请参考 这个页面 予以解决

3.微信出现界面中文字体显示方块的问题

请在终端中输入：

sudo apt-cache search wqy fonts-wqy-microhei - Sans-serif style CJK font derived from Droid fonts-wqy-zenhei - "WenQuanYi Zen Hei" A Hei-Ti Style (sans-serif) Chinese font xfonts-wqy - WenQuanYi Bitmap Song CJK font for X

然后在终端中输入：

sudo apt install fonts-wqy-microhei

4.微信等无法正常收发图片

在终端中输入：

sudo apt install libjpeg62:i386

如有其他问题，请参考官方GITHUB的issues
